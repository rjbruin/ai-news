API Keys

What are API keys?

To cover the cost of the AI models that make this app work, you need to create an account with the website OpenRouter, and provide this app with an API key.

These OpenRouter keys pay for fetching and tagging the news sources you assign them to (see Sources), and one of your own keys can also pay for generating your editions. Revoke a key to stop it being used immediately — any source using it is disabled automatically.

What are API keys?

Dispatch uses AI models to read your news sources and write your editions. Running those models costs real money, charged per request — this app doesn't have a subscription fee, so instead it needs you to bring your own credit to pay for the AI usage your account causes.

OpenRouter is a service that gives you a single account and a single API key that works with models from many different AI providers (Anthropic, OpenAI, Google, and others). You add credit to your OpenRouter account, then give Dispatch an API key — a secret code, like a password, that lets Dispatch spend from that credit on your behalf, without ever needing your actual OpenRouter login.

To create an OpenRouter API key:

  1. Go to openrouter.ai and create an account.
  2. Add some credit to your account (OpenRouter accepts card payments; a few dollars goes a long way for this app).
  3. Go to openrouter.ai/keys and create a new API key.
  4. Copy the key (it starts with sk-or-) and paste it into the "Add a key" form on this page.

Your key is stored encrypted and is only ever used to pay for AI requests your account triggers — nothing else has access to your OpenRouter credit.
