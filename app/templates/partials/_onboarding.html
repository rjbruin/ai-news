Welcome to Dispatch
What Dispatch does

Dispatch watches the news sources you point it at — newsletters, RSS/Atom feeds — pulls out the individual stories, and tags each one against a shared topic taxonomy. An AI agent then writes editions from whatever matches its interests, on a schedule.

Follow some Dispatches

A Dispatch is an AI-written publication; each issue is an edition. Follow any number — their editions show up together on your Editions page. You can change these anytime from the Dispatches page.

Sources are shared, but configurable per person

Everyone using this instance shares the same pool of sources on the Sources page — no need to set up your own newsletter subscription if someone already has. Approved users can add their own sources too.

Don't want a particular source in your own editions? Turn it off just for yourself from the "In my editions" toggle on the Sources page — it stays on for everyone else.

Your edition is written for you

There's nothing to manually curate — the agent selects, groups, and writes up the stories that match your Dispatch's interests every time an edition is cut.

Once you have an edition, you can also export it as a printable PDF or generate a narrated podcast episode of it, if that's enabled for your account.

Or run your own Dispatch

Following a Dispatch is free — someone else pays for generating it. Want one tuned exactly to you instead? Bring your own OpenRouter API key and set up a custom Dispatch of your own in Settings.

It starts as a copy of the System Dispatch's settings, and you shape it from there — then you can publish it for others to follow, too.

⚠️ A custom Dispatch runs on AI models that cost money

Reading sources and writing editions uses AI models, and those cost real money per request. This app doesn't charge a subscription — instead, a custom Dispatch is paid for with your own OpenRouter API key.

When you set up a custom Dispatch, add a key in the API Keys section of Settings — it walks you through creating one, and gives you a rough idea of what it costs.
